# claude-project-manager

A Claude Code plugin marketplace for project management workflows.

## Plugins

| Plugin | Description |
|--------|-------------|
| `proj` | Project lifecycle management: init, status, update, todo, report, archive |

## Planned Skills

- `proj/init` — scaffold a new project with tracking files
- `proj/status` — summarize current project state
- `proj/update` — update project notes and progress
- `proj/todo` — manage project todos
- `proj/report` — generate a project report
- `proj/archive` — archive a completed project
